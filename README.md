# 🦁 Atlas Lions 2026 Predictor

An end-to-end **Machine Learning** web application built with **Python** and **Streamlit** to predict Morocco's national football team match outcomes.

📊 Project Overview
This tool analyzes historical match data (from 2018 to present) to estimate the win probability of the Moroccan National Team based on:

- Opponent Strength: Classified into difficulty levels.
- Team Form: Calculated from recent performances.
- Goal Difference: Assessing offensive/defensive efficiency.

🚀 Tech Stack

- Modeling: Random Forest Classifier (Scikit-Learn).
- Web App: Streamlit.
- Data Handling: Pandas, Numpy.
- Deployment: Streamlit Cloud.

📈 Model Performance
The current model achieves an accuracy of **95.83%** on the testing dataset, demonstrating high reliability in predicting match results based on the "State-of-the-Art" patterns of the Atlas Lions.
